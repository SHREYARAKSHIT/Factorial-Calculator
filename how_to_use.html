How To Use

Factorial(n) is defined recursively as:
Factorial(0)=1, Factorial(1)=1;
Factorial(n)=n*Factorial(n-1);
Just put any non-negative integer or an expression which will return a non-negative integer.
Press Result button to get the result and Reset button to get back to the initial state.
Non-numeric input or Negative number input will lead to "Invalid Input".
